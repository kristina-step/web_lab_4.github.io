# WeatherSphere - Приложение для прогноза погоды

## Описание
Веб-приложение для просмотра прогноза погоды с возможностью добавления нескольких городов.

## Функционал
- Автоматическое определение местоположения
- Добавление до 2+ дополнительных городов
- Прогноз погоды на 3 дня (сегодня + 2 дня)
- Сохранение данных в localStorage
- Адаптивный дизайн для всех устройств
- Кнопка обновления данных


## Использование
1. При первом запуске разрешите доступ к геолокации
2. Для добавления города нажмите кнопку "Добавить город"
3. Выберите город из списка или введите название
4. Переключайтесь между городами в левой панели
5. Обновляйте данные кнопкой "Обновить"

P.S. все отлично работает при включенном VPN, при выключенном не так быстро, долго загружается.
